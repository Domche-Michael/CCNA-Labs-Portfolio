# CCNA Labs Portfolio

This repository showcases hands-on networking labs completed as part of my CCNA training and advanced practice.

## 🔧 Technologies Covered
- VLAN & Inter-VLAN Routing (ROAS, SVI)
- DHCP (IPv4 / Relay)
- NAT & PAT
- ACL (Standard & Extended)
- OSPF (Multi-area)
- EIGRP
- STP (PVST, Rapid-PVST)
- EtherChannel (LACP / PAgP)
- HSRP (Gateway Redundancy)
- Basic Network Security

## 🧪 Lab Categories
- VLAN & Switching
- Routing (OSPF, EIGRP)
- Services (DHCP, NAT)
- Security (ACL)
- Redundancy & High Availability
- Troubleshooting Labs

## 🎯 Objective
To demonstrate real-world networking skills, configuration, and troubleshooting aligned with CCNA-level and beyond.

## 📁 Structure
Each folder contains:
- Packet Tracer labs (.pkt)
- Topology diagrams
- Configuration explanations
