# DHCP

## Objective
Automate IP addressing for IPv4 and IPv6 clients.

## Skills
- DHCPv4 configuration
- DHCP relay (ip helper-address)
- Excluded addresses

## Advanced Concepts
- DHCP server on network server (www.cite.ca)
- DHCPv6 Stateless configuration
- DNS distribution via DHCP
- Multi-VLAN DHCP design

## Labs
- DHCP for VLAN 10 and VLAN 20
- DHCPv6 stateless configuration
