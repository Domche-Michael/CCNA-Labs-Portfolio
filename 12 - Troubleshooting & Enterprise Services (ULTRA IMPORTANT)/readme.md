# Troubleshooting & Enterprise Services

## Objective
Diagnose and manage complex enterprise networks.

## Skills
- Connectivity testing (ping, traceroute)
- Routing table verification
- NAT verification (show ip nat translations)
- BGP verification (show ip bgp)

## Advanced Concepts
- SNMP configuration (monitoring)
- SPAN (traffic capture)
- PPP (PAP & CHAP authentication)
- GRE Tunnel (IPv6 over IPv4)
- eBGP (ISP connectivity)
- Static routing (default routes)
- End-to-end connectivity validation
